По умолчанию участок определяется автоматически по превышению порогового уровня сигнала.

Если участок вырезан неверно, попробуйте изменить параметры в начале блока:

  • измените порог отклонения сигнала threshold (в единицах стандартного отклонения);
  • Задайте вручную переменные MANUAL_START и MANUAL_STOP (в секундах).

ВНИМАНИЕ! Убедитесь, что максимумы и минимумы на графике ниже сменяют друг друга последовательно (не перемежаются). Если последовательность нарушена, переделайте опыт (возможно, с другим магнитом).

6. Аппроксимация зависимости и определение ускорения¶

Теоретическая зависимость координаты груза от времени имеет вид $x(t) = v_0 t + a t^2 /2$ (момент $t=0$ соответствует первому пику).

Численным методом наименьших квадратов (метод curve_fit) подбираются оптимальные парметры $v_0$ и $a$, минимизирующие сумму квадратов отклонений экспериментальных точек от теоретической кривой.

Начальные скорости по максимуму и минимуму сигналов будут разными (так как соответствуют разным положениям магнита), однако ускорения должны быть одинаковы в пределах погрешности.

In [116]:
motion_eq = lambda t, v0, a: v0 * t + 0.5 * a * t**2  # уравнение движения

coeffs_h, err_h = curve_fit(motion_eq, highs, X) # подбор по максимумам
coeffs_l, err_l = curve_fit(motion_eq, lows, X)  # подбор по минимумам

v1, a1 = coeffs_h
sa1 = np.sqrt(err_h[1][1])
v2, a2 = coeffs_l
sa2 = np.sqrt(err_l[1][1])

# график
plt.figure(figsize = (8, 6))
plt.title("Аппроксимация закона движения")
t = np.linspace(0., 1.1 * max(highs[-1], lows[-1]), 100)
plt.plot(t, motion_eq(t, *coeffs_h), color="blue", 
         label="Аппроксимация %.3f $t$ + %.3f $t^2$/2" % (v1, a1))
plt.plot(highs, X, '^', label="Измерения (максимумы)")
plt.plot(t, motion_eq(t, *coeffs_l), color="orange", 
         label="Аппроксимация %.3f $t$ + %.3f $t^2$/2" % (v2, a2))
plt.plot(lows, X, 'v', label="Измерения (минимумы)")
plt.legend()
plt.ylabel("Координата катушки $x$, м")
plt.xlabel("Время $t$, с")
plt.grid()
plt.show()

print("Ускорение по максимумам / по минимумам:")
print(f"a_max = {a1:.3f} ± {sa1:.3f}, a_min = {a2:.3f} ± {sa2:.3f} [м/с^2]")
print("Среднее:")
print(f"<a> = {0.5 * (a1 + a2):.3f} ± {0.5 * np.sqrt(sa1**2 + sa2**2):.3f} [м/с^2]")

# перевод углов в радианы
theta_rad = theta / 180 * np.pi
sigma_theta_rad = sigma_theta / 180 * np.pi

# замена переменных для линеаризации графика
ap = a / np.cos(theta_rad)
tn = np.tan(theta_rad)

# пересчёт погрешностей точек
sigma_cos = np.abs(np.sin(theta_rad)) * sigma_theta_rad
sigma_tan = sigma_theta_rad / np.cos(theta_rad)**2
sigma_ap = ap * np.sqrt((sigma_a / a)**2 + (sigma_cos / np.cos(theta_rad))**2)

f = lambda x, k, b: k * x + b

K, err = curve_fit(f, tn, ap, sigma=sigma_ap)

# результат аппроксимации
mu = -K[1]/K[0]
g = K[0]

# погрешности результа
sigma_g = np.sqrt(err[0][0])
sigma_mu = mu * np.sqrt(err[0][0] / g**2 + err[1][1] / K[1]**2) 

print(f"µ = {mu:.3f} +- {sigma_mu:.3f} ; g = {g:.3f} +- {sigma_g:.3f} м^2/с")

plt.figure(figsize = (9, 6))
plt.title("Аппроксимация зависимости ускорения от угла наклона")
plt.errorbar(tn, ap, xerr=sigma_tan, yerr=sigma_ap, fmt='ko', ms=3, elinewidth=1, capsize=2)
plt.xlabel(r"$\tan \theta$")
plt.ylabel(r"$a\: /\: \cos \theta$")
plt.xlim(0, 1.1 * np.max(tn))
plt.ylim(0, 1.1 * np.max(ap))
plt.plot(tn, f(tn, *K), '-')
